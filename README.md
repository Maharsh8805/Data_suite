# Data Suite

A simple and powerful tool to clean and process various data formats including CSV, Text, Images, Audio, and Video.

## Features

- **CSV Cleaner**: Clean tabular data, handle missing values, and remove outliers.
- **Text Cleaner**: Sanitize text, remove noise, and handle encoding issues.
- **Image Cleaner**: Resize, filter, and clean image metadata.
- **Audio Studio**: Analyze and process audio files.
- **Video Studio**: Process video files, including denoising and trimming.

## Installation

1. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

1. Run the main application:
   ```bash
   streamlit run main_app.py
   ```
2. Open your browser to `http://localhost:8501`.
